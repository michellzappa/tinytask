### Description (4000 chars max)

TinyTask is a fast, keyboard-first task manager for macOS. Your tasks live in plain Markdown files — no proprietary database, no cloud sync, no account. Open a file, manage your tasks, and close it. The file is yours forever.

Features:
- Plain-text Markdown task lists you can read anywhere
- Keyboard-first design — navigate, complete, and create tasks without touching the mouse
- Sections (# Heading) for organizing tasks into groups
- Tasks (- [ ]) with done (- [x]) and cancelled (- [-]) states
- Checklists — indent tasks to create sub-items under any task
- Notes — add context to any task with indented annotations
- Due dates (@due), start dates (@start), and automatic completion timestamps
- Overdue task highlighting in red
- Per-section progress display
- Status bar with total, done, cancelled, and overdue counts
- Auto-save on every change
- Quick open file finder (Cmd+P)
- Multiple independent windows
- Light and dark mode — follows your system
- Opens .md, .txt, and .todo files directly from Finder

Keyboard shortcuts:
- Arrow keys to navigate, Space to toggle, Return to edit
- Cmd+Return to add a new task, Cmd+Shift+Return for a new section
- Cmd+Up/Down to reorder, Tab to indent into a checklist
- Cmd+. to cancel a task, Cmd+Backspace to delete
- On-device AI assistant (Cmd+K) — ask questions, generate tasks, summarize. Runs via CoreML, no API key required

Built entirely with native macOS technologies. No Electron. No sync services. Just SwiftUI managing your plain-text files, fast and private.

Part of TinySuite — a family of native macOS micro-tools that each do one thing well. Every app is built around the file, not around itself. No accounts, no cloud, no lock-in. Open your file, understand it, act on it. Learn more at tinysuite.app.
